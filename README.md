# 10-minute-test
10-minute-test is a test android application for completing the Android Developer Requirement Test given by Robi Ten Minute School.  
 
# Features!

  - Item List
  - Expandable-collapsible Item
  - Sub Items
  - Video Layout
  - Audio Layout
 
### Technology

* MVVM
*  JAVA
*  LiveData
*  Data Binding
*  Material Design
